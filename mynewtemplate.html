Throughout the semester in Client Side Web Development, I have learned the ins and outs of html. I started this course with no prior experience or knowledge in web development, though I have made great strides throughout the course of the semester. Early on, I learned how to create a standard html document using notepad++ with the head and body portion. I utilized standard tags like heading, linebreak, bold, and more. I then got into ordered lists, unordered lists, and a definition list. More importantly though, I got familiar with how to upload onto my own personal webspace. I then created my own sitemap and learned how to place images onto my webpage. This is also around the time I learned about navigation and setting up relative/local links, named anchor/targeted links, and contextual links to my pages. A few months into the course, I started to learn about tables and tabular data. I also got an introduction to css which taught me how to build a webpage that was not only visually appealing, but functional as well.

One of the bigger assignments that I learned a great deal from was building a page layout template. I had to take a lot of what I learned and put it into practice by creating a template with 4 stacking tables, 2 complete navigation structures using relative links, and css for styling. After completing the template, I was also tasked with migrating my work using the new template. I learned how to compile my work into a polished website.
