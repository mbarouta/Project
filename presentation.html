Two Minute Presentation on:

 

<bdi>

What is the purpose/use of the <bdi> element?

The <bdi> (Bidirectional Text Isolation) HTML semantic/structural element instructs the browser’s bidirectional algorithm to treat the text it contains in isolation from the surrounding text. The element is commonly used when websites dynamically insert text without knowing the directionality of which the text is being inserted. Bidirectional text includes both left-to-right and right-to-left sequences of text. A great example of a right-to-left sequence of text that we may find unusual is Arabic text. Since Arabic text is written right-to-left, it may cause issues when embedding it into an English string. Not to worry though, the purpose/use of the <bdi> element is to correct the issues that may come up when working with differing directionalities of text.

Which browsers support the use of the <bdi> element?

  1. Google Chrome
  2. Mozilla Firefox
  3. Microsoft Edge
  4. Safari
  5. Opera

Internet Explorer does not support the use of the <bdi> tag.

Did the <bdi> element replace an existing HTML element?

The <bdi> element did not replace an existing HTML element.
It was introduced to address a specific need.

Here is an example of how the <bdi> element can be used :

Here I will show my name in Arabic followed by my age without the use of the <bdi> tag :

  • مارك: 24 years old

Here I will show my name in Arabic followed by my age with the use of the <bdi> tag :

  • مارك: 24 years old
  • (Translation) Mark: 24 years old
